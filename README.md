# Easynotes

Easynotes is a unique cross-platform notes web app built using ReactJS

### Features
- Add Notes
- Edit Notes
- Delete Notes
- Markdown Enabled Typing
- Search For Notes
- Auto Save [Saves in localstorage in browser- Instant delete button provided]
- Change Color of Notes
- Dark Mode
